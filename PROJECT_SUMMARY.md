# Solonest 项目总结

## 📱 项目概述

**Solonest** 是一个面向独立创业者和 Solo 创业者的社交平台，以微信小程序形式呈现。

**品牌标语**: "Connect, share, thrive"  
**设计关键词**: warm, hopeful, happy

---

## 🎨 品牌设计

### 色彩系统
- **主色 (Primary)**: `#FFB347` - 温暖橙
- **辅色 (Secondary)**: `#81D4FA` - 信任蓝
- **强调色 (Accent)**: `#CE93D8` - 创意紫

### 字体
- **主字体**: Comfortaa (品牌标题)
- **正文字体**: Open Sans (英文) / 微信默认字体 (中文)

### 设计风格
- 活泼温暖的多彩标签系统
- 渐变色头像边框
- 彩色爱好图标
- 圆润的卡片设计

## 🛠 技术栈

### 微信小程序
- **框架**: 微信小程序原生框架
- **语言**: TypeScript
- **样式**: WXSS (类 CSS)

### 为什么选择原生小程序而不是 Taro？

**原因**:
1. Taro CLI 初始化遇到技术问题
2. 原生小程序更稳定，性能更好
3. 对于小程序专属项目，原生开发更直接
4. 微信开发者工具支持完善

**优势**:
- ✅ 无需额外编译步骤
- ✅ 完全兼容微信小程序 API
- ✅ 开发调试更直观
- ✅ 包体积更小

## 📋 核心功能规范

### 用户信息字段

**必填项**:
- 我能提供的（每行最多50字，最多10行）
- 我正在做（每行最多50字，最多5行）
- 合作方式偏好（最多100字）

**选填项**:
- 个人爱好（最多6个图标）
- 个人技能（AI 生成标签）
- 照片（最多5张）
- 工作经验（最多500字，AI 生成标签）
- 补充资料（最多3张图片）

### 特殊规则

1. **需求发布**: 必须关联"我正在做"中的一条项目
2. **评价权限**: 聊天消息超过10条后才能评价
3. **在线状态**: 打开小程序为在线，切换后台5分钟内仍显示在线
4. **关注功能**: 直接关注，无需对方同意

## 📖 如何使用

### 1. 查看设计文档
- 阅读 `prd.md` 了解产品功能
- 阅读 `ui-design-spec.md` 了解设计规范

### 2. 查看原型设计
- 打开 `stitch_/` 目录下的 HTML 文件
- 在浏览器中查看页面设计

### 3. 运行小程序
1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具
3. 导入 `miniprogram` 目录
4. 选择"测试号"或输入 AppID
5. 点击"编译"查看效果
