# Solonest 微信小程序

独立创业者社交平台 - 微信小程序版本

## 项目结构

```
miniprogram/
├── pages/              # 页面目录
│   └── profile/        # 个人主页
│       ├── profile.wxml    # 页面结构
│       ├── profile.wxss    # 页面样式
│       ├── profile.ts      # 页面逻辑
│       └── profile.json    # 页面配置
├── components/         # 组件目录
├── utils/             # 工具函数
├── assets/            # 静态资源
├── app.json           # 小程序配置
├── app.wxss           # 全局样式
├── app.ts             # 小程序入口
└── project.config.json # 项目配置
```

## 技术栈

- **框架**: 微信小程序原生框架
- **语言**: TypeScript
- **样式**: WXSS (类 CSS)

## 设计规范

### 品牌色系
- 主色 (Primary): `#FFB347` - 温暖橙
- 辅色 (Secondary): `#81D4FA` - 信任蓝
- 强调色 (Accent): `#CE93D8` - 创意紫

### 文字颜色
- 主要文字: `#333333`
- 次要文字: `#666666`
- 辅助文字: `#999999`

### 圆角规范
- 小卡片: `16rpx` (8px)
- 中卡片: `32rpx` (16px)
- 大卡片: `40rpx` (20px)
- 照片/图片: `24rpx` (12px)

### 标签样式
- 技能类: 浅蓝背景 `#E3F2FD` / 深蓝文字 `#1976D2`
- 资源类: 浅绿背景 `#E8F5E9` / 深绿文字 `#388E3C`
- 合作类: 浅紫背景 `#F3E5F5` / 深紫文字 `#7B1FA2`
- 个人类: 浅橙背景 `#FFF3E0` / 橙色文字 `#FF9800`

## 开发指南

### 1. 安装微信开发者工具

下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 2. 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `miniprogram` 目录
4. AppID 选择"测试号"或输入你的 AppID

### 3. 编译运行

点击"编译"按钮即可在模拟器中预览

### 4. 真机调试

点击"预览"按钮，使用微信扫码在真机上调试

## 页面说明

### 个人主页 (profile)

展示用户的个人信息和商业信息，包括：

**个人信息区域**:
- 爱好（彩色图标展示）
- 技能（多彩标签）
- 照片（网格展示，最多5张）

**商业信息区域**:
- 我能提供的（必填，AI 生成标签）
- 我正在做（必填）
- 合作方式偏好（必填）
- 工作经验（选填，AI 生成标签）
- 补充资料（最多3张图片）

**其他功能**:
- 关注/取消关注
- 发起聊天
- 举报

## 注意事项

1. 本项目使用 TypeScript，需要在微信开发者工具中启用 ES6 转 ES5
2. 图片资源需要替换为实际的图片 URL
3. 需要配置后端 API 接口
4. 上线前需要申请正式的 AppID
