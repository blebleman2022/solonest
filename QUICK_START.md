# Solonest 快速开始指南

## 🚀 5分钟快速体验

### 步骤 1: 安装微信开发者工具

1. 访问 [微信开发者工具下载页](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 根据你的操作系统下载对应版本
3. 安装并打开微信开发者工具

### 步骤 2: 导入项目

1. 打开微信开发者工具
2. 点击左侧的 **"+"** 或 **"导入项目"**
3. 填写项目信息：
   - **项目名称**: Solonest
   - **目录**: 选择本项目的 `miniprogram` 文件夹
   - **AppID**: 选择 **"测试号"**（或输入你的 AppID）
4. 点击 **"导入"**

### 步骤 3: 查看效果

1. 项目导入后会自动编译
2. 在左侧模拟器中即可看到个人主页效果
3. 可以点击各个元素测试交互功能

### 步骤 4: 真机预览（可选）

1. 点击工具栏的 **"预览"** 按钮
2. 使用微信扫描生成的二维码
3. 在手机上查看真实效果

## 📱 页面功能说明

### 个人主页包含以下区域：

#### 1️⃣ 头部区域
- 渐变色背景（橙→蓝）
- 头像（带渐变边框）
- 昵称、位置、在线状态
- 举报按钮

#### 2️⃣ 个人标签
- 横向滚动的标签列表
- 从"我能提供的"AI 提取

#### 3️⃣ 关注数据
- 关注数（橙色强调）
- 受关注数（橙色强调）

#### 4️⃣ 个人信息区域
- **爱好**: 彩色图标展示（最多6个）
- **技能**: 多彩标签展示
- **照片**: 网格展示（最多5张，可点击预览）

#### 5️⃣ 商业信息区域
- **我能提供的** ⭐必填
  - 列表展示
  - AI 生成标签
- **我正在做** ⭐必填
  - 列表展示
- **合作方式偏好** ⭐必填
  - 文字描述
- **工作经验**
  - 文字描述
  - AI 生成标签
- **补充资料**
  - 图片展示（最多3张，可点击预览）

#### 6️⃣ 合作评价
- 显示其他用户的评价
- 当前为空状态

#### 7️⃣ 底部操作栏
- **关注** 按钮（轮廓样式）
- **发起聊天** 按钮（主色样式）

## 🔧 自定义数据

### 修改示例数据

打开 `miniprogram/pages/profile/profile.ts` 文件，找到 `data` 对象，修改以下字段：

```typescript
data: {
  userInfo: {
    avatar: '你的头像URL',
    nickname: '你的昵称',
    location: '你的位置',
    // ... 其他字段
  }
}
```

### 主要字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `avatar` | string | 头像图片 URL |
| `nickname` | string | 用户昵称 |
| `location` | string | 所在城市 |
| `onlineStatus` | string | 在线状态（在线/离线） |
| `tags` | string[] | 个人标签数组 |
| `followingCount` | number | 关注数 |
| `followersCount` | number | 受关注数 |
| `hobbies` | object[] | 爱好数组（icon, label, bgColor） |
| `skills` | string[] | 技能数组 |
| `photos` | string[] | 照片 URL 数组（最多5张） |
| `canProvide` | string[] | 我能提供的（最多10条） |
| `currentProjects` | string[] | 我正在做（最多5条） |
| `collaborationPreference` | string | 合作方式偏好 |
| `workExperience` | string | 工作经验 |
| `additionalMaterials` | string[] | 补充资料 URL 数组（最多3张） |

## ❓ 常见问题

### Q: 为什么图片显示不出来？
A: 示例数据使用的是占位图片 URL (`https://via.placeholder.com/`)，需要替换为真实的图片 URL。

### Q: 如何修改品牌颜色？
A: 修改 `miniprogram/app.wxss` 文件中的颜色变量。

### Q: 如何添加更多页面？
A: 
1. 在 `miniprogram/pages/` 下创建新的页面文件夹
2. 在 `miniprogram/app.json` 的 `pages` 数组中添加页面路径

### Q: 如何连接后端 API？
A: 在页面的 `.ts` 文件中使用 `wx.request()` 调用后端接口。
